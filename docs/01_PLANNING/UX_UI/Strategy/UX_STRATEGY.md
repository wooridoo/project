## 1. UX 비전

### 핵심 메시지

> **"소모임처럼 모이고, 토스처럼 관리하되, 먹튀 걱정은 없다"**

### 비전 선언문

WOORIDO는 **신뢰 기반의 커뮤니티 금융 경험**을 제공합니다.
- 취미와 관심사를 공유하는 사람들이 **함께 모으고, 투명하게 사용**할 수 있는 플랫폼
- 금전적 불안을 해소하고, 모임의 본질인 **관계와 활동**에 집중할 수 있는 환경

## 2. Trust Triangle (신뢰 삼각형)

```
                    △
                   /|\
                  / | \
                 /  |  \
                /   |   \
    ┌──────────┐ ┌──┴──┐ ┌──────────┐
    │ 선충전 락 │ │ 장부 │ │ 결제 감시 │
    │ Deposit  │ │ 투명화│ │Consensus │
    │   Lock   │ │     │ │   Pay    │
    └──────────┘ └─────┘ └──────────┘

     1개월 보증금    1원 단위     모든 지출
     선입금으로     투명 공개     멤버 투표
     이탈 방지                   승인 필요
```

### Trust Triangle UX 표현

| 요소 | UX 표현 | 위치 |
|------|---------|------|
| **선충전 락** | 🔒 아이콘 + "보증금 보호" 뱃지 | 가입 화면, 프로필 |
| **장부 투명화** | 📊 Recharts 차트 + 실시간 잔액 | 장부 탭 |
| **결제 감시** | 🗳️ 투표 승인 플로우 + 참석자 표시 | 투표 탭, 지출 요청 |

## 4. UX 원칙 (5가지)

### 원칙 1: 투명성 (Transparency)

> 모든 금전 흐름은 1원 단위까지 공개한다.

**적용 예시**:
- 장부 탭: 모든 거래 내역 타임라인
- 입회비 계산: 계산식 투명 공개
- 지출 영수증: 상호명 자동 파싱 (수정 불가)

### 원칙 2: 민주성 (Democracy)

> 돈이 움직이려면 멤버의 동의가 필요하다.

**적용 예시**:
- 지출 투표: 과반수~70% 승인 필요
- 강퇴 투표: 70% 승인 필요
- 해산 투표: 100% 전원 동의

### 원칙 3: 신뢰성 (Trust)

> 시스템이 신뢰를 보장하여 먹튀 걱정을 없앤다.

**적용 예시**:
- 보증금 락: 가입 시 1개월분 선입금
- 과반수 참석: 모임 개최 필수 조건
- Brix 당도: 신뢰도 시각화

### 원칙 4: 접근성 (Accessibility)

> 누구나 쉽게 이해하고 사용할 수 있다.

**적용 예시**:
- 친근한 용어: 챌린지, 서포트, 베네핏
- 툴팁: 복잡한 개념 설명
- 반응형: 모바일 우선, 데스크탑 지원

### 원칙 5: 즐거움 (Delight)

> 금융 경험도 즐거울 수 있다.

**적용 예시**:
- Brix 과일 이모지: 🍯🍇🍎🍊🍅🥒
- 완주 축하: Confetti 애니메이션
- 마이크로 인터랙션: 호버, 탭 피드백
