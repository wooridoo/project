# WOORIDO UX 인터랙션 (UX Interactions)

> **Version**: 1.0.0
> **Last Updated**: 2026-01-16
> **Purpose**: 공통 인터랙션 패턴 및 제스처 정의

## 1. 공통 인터랙션 패턴

### 1.1 Pull to Refresh

| 항목 | 스펙 |
|------|------|
| **적용 화면** | 홈, 피드, 장부, 투표, 모임 |
| **트리거** | 스크롤 최상단에서 아래로 당기기 (60px 이상) |
| **피드백** | Spinner + 진동 (10ms) |
| **완료** | Spinner 숨김 + 컨텐츠 갱신 |
| **실패** | Toast "새로고침 실패. 다시 시도해주세요" |

### 1.2 Infinite Scroll

| 항목 | 스펙 |
|------|------|
| **적용 화면** | 피드, 장부 타임라인, 검색 결과 |
| **트리거** | 스크롤 90% 도달 |
| **피드백** | 하단 Spinner |
| **완료** | 새 아이템 추가 |
| **마지막 페이지** | "더 이상 항목이 없습니다" |

### 1.3 Swipe Actions

| 방향 | 화면 | 액션 | 결과 |
|------|------|------|------|
| **← 좌측** | 알림 아이템 | 삭제 | 빨간색 배경 + 🗑️ |
| **→ 우측** | 알림 아이템 | 읽음 처리 | 파란색 배경 + ✓ |
| **← 좌측** | 게시글 카드 | (없음) | - |

### 1.4 Long Press

| 화면 | 대상 | 액션 | 결과 |
|------|------|------|------|
| 피드 | 게시글 | 500ms | 컨텍스트 메뉴 (수정/삭제/신고) |
| 피드 | 댓글 | 500ms | 컨텍스트 메뉴 |
| 장부 | 거래 항목 | 500ms | 상세 모달 |

## 2. 탭 인터랙션

### 2.1 단일 탭

| 대상 | 결과 |
|------|------|

| **카드** | 상세 화면 이동 |

| 대상 | 결과 |
|------|------|
| **게시글** | 좋아요 토글 + ❤️ 애니메이션 |
| **이미지** | 줌인 |

## 3. 폼 인터랙션

### 3.1 입력 필드

| 상태 | 시각적 피드백 |
|------|-------------|
| **Default** | Grey border |
| **Focus** | Orange border (2px) + 라벨 상단 이동 |
| **Filled** | Grey border + 값 표시 |
| **Error** | Red border + 에러 메시지 |
| **Disabled** | Grey background + not-allowed cursor |

### 3.2 버튼 상태

| 상태 | 시각적 피드백 |
|------|-------------|
| **Default** | 기본 색상 |
| **Hover** | 밝기 -10% |
| **Active/Pressed** | 밝기 -20% + scale(0.98) |
| **Loading** | Spinner + disabled |
| **Disabled** | opacity 50% |

### 3.3 유효성 검증

| 시점 | 검증 타입 |
|------|----------|
| **onBlur** | 개별 필드 검증 |
| **onChange** | 실시간 검증 (비밀번호 강도) |
| **onSubmit** | 전체 폼 검증 |

## 4. 모달 인터랙션

### 4.1 열기/닫기

| 액션 | 애니메이션 |
|------|-----------|
| **열기** | fade-in + slide-up (300ms) |
| **닫기** | fade-out + slide-down (200ms) |

### 4.2 닫기 트리거

| 방법 | 적용 모달 |
|------|----------|
| **X 버튼** | 모두 |
| **배경 탭** | 정보성 모달만 (결제/확인 모달 제외) |
| **스와이프 다운** | BottomSheet |
| **ESC 키** | 데스크탑 |

### 4.3 BottomSheet 특수 동작

| 높이 | 동작 |
|------|------|
| **Peek (30%)** | 스와이프 업 → 확장 |
| **Half (50%)** | 기본 열린 상태 |
| **Full (90%)** | 최대 확장 |
| **스와이프 다운** | 현재 단계 아래로 축소/닫기 |

## 5. 네비게이션 인터랙션

### 5.1 탭 전환

| 방법 | 피드백 |
|------|--------|
| **탭 탭** | 즉시 전환 + 인디케이터 이동 |
| **스와이프** | 제스처 추적 + 탭 전환 |

### 5.2 뒤로 가기

| 플랫폼 | 방법 | 동작 |
|--------|------|------|
| **Mobile** | 좌측 스와이프 | 이전 화면 |
| **Mobile** | 하드웨어 백 | 이전 화면 |
| **Desktop** | 브라우저 뒤로 | 이전 화면 |

## 6. 접근성 고려

### 6.1 터치 타겟

| 요소 | 최소 크기 |
|------|----------|
| **버튼** | 44px × 44px |
| **아이콘 버튼** | 48px × 48px |
| **리스트 아이템** | 높이 48px+ |

### 6.2 키보드 네비게이션

| 키 | 동작 |
|----|------|
| **Tab** | 다음 포커스 요소 |
| **Shift+Tab** | 이전 포커스 요소 |
| **Enter** | 버튼 클릭/제출 |
| **Space** | 체크박스 토글 |
| **Esc** | 모달 닫기 |
| **Arrow** | 드롭다운/라디오 선택 |

### 6.3 스크린 리더

| 요소 | aria 속성 |
|------|----------|
| **버튼** | aria-label |
| **이미지** | alt |
| **모달** | aria-modal, aria-labelledby |
| **로딩** | aria-busy |
| **에러** | aria-invalid, aria-describedby |
