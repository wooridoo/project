## 1. 당도 시스템 개요

### 1.1 컨셉

WOORIDO의 신뢰 지표를 **"당도(Brix)"**로 표현합니다.

- **🍯 달콤한 관계**: 당도가 높을수록 신뢰할 수 있는 회원
- **🥒 쓴 관계**: 당도가 마이너스면 문제가 있는 회원

### 1.2 당도 범위

| 구분 | 당도 | 의미 |
|------|------|------|
| 최대 | 80 | 꿀처럼 달콤한 관계 |
| 시작 | 12 | 맛있는 과일 수준 (기본 신뢰) |
| 최저 | 없음 | 마이너스 가능 (쓴맛) |

## 2. 당도 체계

### 2.1 기본 당도

| 구분 | 당도 |
|------|------|
| 가입 시 기본 부여 | **12** |

### 2.2 납입 당도

| 항목 | 당도 | 설명 |
|------|------|------|
| 모임 참석 횟수 당 | +0.09 | 오프라인/온라인 모임 참석 시 |
| 납입 개월수 | +0.32 | 정상 납입 완료 월 기준 |
| 연체 횟수 당 | -1.5 | 납입 기한 초과 시 |
| 연속 연체 (2회 이상) | -1.0 | 기존 감점에 추가 |

### 2.3 활동 당도

| 항목 | 당도 | 일일 제한 | 월간 최대 |
|------|------|----------|----------|
| 피드 작성 당 | +0.05 | 1개/일 | 30개 (1.5) |
| 댓글 작성 당 | +0.025 | 3개/일 | 90개 (2.25) |
| 좋아요 당 | +0.006 | 5개/일 | 150개 (0.9) |
| 리더 경험 1개월 당 | +0.45 | - | 0.45 |
| 투표 불참 당 | -0.1 | - | 모임 투표 미참여 시 |
| 신고 당함 횟수 당 | -0.6 | - | 동일인 다수 신고는 1회 처리 |
| 강퇴 당함 횟수 당 | -4.0 | - | - |

### 2.4 배율 및 계산식

| 구분 | 값 |
|------|------|
| 납입 당도 배율 | 0.7 |
| 활동 당도 배율 | 0.15 |
| 당도 상한 | 80 |
| 당도 하한 | 없음 (마이너스 가능) |

**최종 당도 계산식:**

```
최종 당도 = 기본 당도(12) + (납입 당도 × 0.7) + (활동 당도 × 0.15)
```

## 5. 1년 시나리오 상세 계산

### 5.1 A 유형 (모범 리더)

**납입 당도 계산:**

| 항목 | 계산식 | 당도 |
|------|--------|------|
| 모임 참석 | 6회 × 12개월 × 0.09 | 6.48 |
| 납입 개월 | 3개 챌린지 × 12개월 × 0.32 | 11.52 |
| 연체 | 0회 × -1.5 | 0 |
| 연속 연체 | 0회 × -1.0 | 0 |
| **납입 합계** | | **18.0** |

**활동 당도 계산:**

| 항목 | 계산식 | 당도 |
|------|--------|------|
| 피드 | 25개 × 12개월 × 0.05 | 15.0 |
| 댓글 | 75개 × 12개월 × 0.025 | 22.5 |
| 좋아요 | 130개 × 12개월 × 0.006 | 9.36 |
| 리더 | 12개월 × 0.45 | 5.4 |
| 투표 불참 | 0회 × -0.1 | 0 |
| **활동 합계** | | **52.26** |

**최종 당도:**
```
12 + (18.0 × 0.7) + (52.26 × 0.15) = 12 + 12.6 + 7.84 = 32.44
```

### 5.2 B 유형 (적극 참여자)

- **납입 당도**: 12.0
- **활동 당도**: 37.2

**최종 당도:**
```
12 + (12.0 × 0.7) + (37.2 × 0.15) = 12 + 8.4 + 5.58 = 25.98
```

### 5.3 C 유형 (보통 사용자)

- **납입 당도**: 7.32
- **활동 당도**: 21.78

**최종 당도:**
```
12 + (7.32 × 0.7) + (21.78 × 0.15) = 12 + 5.12 + 3.27 = 20.39
```

### 5.4 D 유형 (소극 사용자)

- **납입 당도**: -1.68
- **활동 당도**: 5.04

**최종 당도:**
```
12 + (-1.68 × 0.7) + (5.04 × 0.15) = 12 - 1.18 + 0.76 = 11.58
```
