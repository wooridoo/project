## 📋 용어 변경 배경

### 법적 이슈 회피
- **"계모임"** → 금융 관련 법규 저촉 가능성
- **"계주"** → 금융 거래 주체로 오인 가능
- **"곗돈"** → 상호금융법 저촉 가능성
- **목표**: 커뮤니티 챌린지 플랫폼으로 재포지셔닝

### 서비스 재정의

```
기존: 금융 중심 계모임 서비스
변경: 커뮤니티 챌린지 플랫폼 (금융 기능 포함)
```

## 🏦 계(契)의 유형 구분

### 분배형 계 (Traditional Gye)
| 특성 | 설명 |
|------|------|
| **성격** | 전통적인 계 |
| **목적** | 순번제로 목돈 분배 |
| **방식** | 투자/저축형 |
| **종료** | 순번 완료 시 해체 |
| **우리두 해당** | ❌ |

### 모임형 계 (Modern Gye) - 우리두
| 특성 | 설명 |
|------|------|
| **성격** | 현대적인 계 |
| **목적** | 여행/사교 모임 회비 관리 |
| **방식** | 구독형 (지속적 납입) |
| **종료** | ❌ 해체 개념 없음 (계속 운영) |
| **우리두 해당** | ✅ |

> 💡 **우리두는 모임형 계입니다.**
> 투자 목적이 아닌 사교/여행 모임의 회비를 관리하는 구독형 서비스입니다.

## 🔄 핵심 용어 매핑

### 법적 리브랜딩 용어

| 기존 용어 | 변경 후 | 영문 | DB 컬럼명 | 설명 |
|----------|--------|------|----------|------|
| **계모임** | **챌린지** | Challenge | `challenge` | 그룹 채널 개념 |
| **계모임 내 전체 인원** | **멤버** | Member | `member` | 챌린지 내 가입자 |
| **계주** | **리더** | Leader | `leader_id` | 챌린지 생성자 |
| **계원** | **팔로워** | Follower | `follower` | 챌린지 참여자 |
| **모임** | **정기 모임** | Regular Meeting | `meeting` | 실제 오프라인 이벤트 |
| **보증금** | **보증금** | Deposit | `deposit` | 변경 없음 (안심결제용 락) |

### 재화 및 거래 용어 (NEW)

| 용어 | 영문 | 정의 | 용례 |
|------|------|------|------|
| **크레딧** | Credit | 우리두 가상머니/재화 | "어카운트에 10만 크레딧이 있습니다" |
| **서포트** | Support | 납입 행위 (와디즈 펀딩처럼) | "이번 달 서포트가 완료되었습니다" |
| **베네핏** | Benefit | 리더가 받는 할인 혜택/보상 | "리더 베네핏 5% 할인 적용" |
| **오픈** | Open | 정기 모임 지출에 대해 동의/비동의를 받는 **투표 행위** | "정기 모임 오픈하겠습니다" = "지출 투표 받겠습니다" |
| **챌린지 어카운트** | Challenge Account | 계모임 통장 (서포트 + 입회비가 모이는 곳) | "챌린지 어카운트 잔액: 300만원" |
| **입회비** | Entry Fee | 신규 가입 시 납부하는 공정성 비용 | "입회비 = 잔액 / 팔로워 수" |

> 💡 **용어 구분**
> - **오픈**: 지출에 대한 투표 행위 (동의/비동의)
> - **챌린지 어카운트**: 서포트 + 입회비가 모이는 계모임 통장, 오픈 투표 승인 시 여기서 차감
> - **입회비**: 챌린지 어카운트 잔액 / 팔로워 수 (리더 제외), 신규 가입자의 공정 기여금

> ⚠️ **"리워드" 용어 폐기**
> 기존에 사용하던 "리워드" 용어는 보증금 반환과 혼동을 일으켜 폐기합니다.
> - ❌ "챌린지 리워드: 완주 시 보증금 반환" (잘못된 표현)
> - ✅ "서포트 완료" / "베네핏 적용" (올바른 표현)

## 💰 보증금 시스템 (Deposit Lock)

### 개념
보증금은 **예치금** 개념으로, 미납 시 회비를 충당하기 위한 안전장치입니다.
고시원 보증금이 월세 미납 시 충당되는 것과 동일한 원리입니다.

### 작동 방식

```
[가입 시]
└─ 1개월분 보증금이 개인 어카운트에서 "락" 상태로 전환
└─ 실제로 어카운트에서 빠져나가지 않음
└─ 챌린지 금고(오픈)에 합산되지 않음

[정상 상태]
└─ 개인 어카운트 UI: "10만 크레딧 중 10만 크레딧 락됨"
└─ 락된 금액은 출금/사용 불가

[미납 발생 시]
└─ 보증금에서 자동으로 서포트 납입
└─ "보증금 충전이 필요합니다" 안내 노출
└─ 일정 기간 미충전 시 → 자동 탈퇴 (리더만 알림)

[정상 탈퇴 시]
└─ 보증금 락 해제 → 가용 크레딧으로 복귀
```

### 중요 포인트

| 항목 | 설명 |
|------|------|
| **위치** | 개인 어카운트 내 잠금 상태 |
| **챌린지 금고 합산** | ❌ (오픈에 포함되지 않음) |
| **완주와 관계** | ❌ 무관 |
| **반환 시점** | 탈퇴 시 (미납 없으면 전액) |
| **몰수 조건** | 미납 상태로 탈퇴 시 |

## 🏆 완주 시스템 (Completion)

### 개념
완주는 **계 해체가 아닌 인증**입니다.
인스타그램 블루체크처럼 신뢰성을 증명하는 마크를 부여합니다.

### 완주 조건
- **1년 연속 운영**
- 미납 이력과 무관

### 완주 효과
- 챌린지에 **인증 마크** 부여 (블루체크 컨셉)
- 계는 계속 운영됨 (해체 ❌)
- 서포트(납입) 지속됨

### 인증 마크 대상

| 대상 | 현재 | 향후 (DAU 증가 시) |
|------|------|-----------------|
| **챌린지** | ✅ 인증 마크 부여 | ✅ |
| **리더** | ❌ | ✅ 리더 인증 마크 |
