WOORIDO 용어 사전 (Glossary)
버전: 1.0.0 작성일: 2026-01-16 기준 문서: TERMINOLOGY.md v2.1, PRODUCT_AGENDA.md v4.2, DB_Schema_1.0.0.md, POLICY_DEFINITION.md SSOT 원칙: DB Schema > Policy > API Schema > 개별 도메인 문서

1. 문서 개요
1.1 목적
본 문서는 WOORIDO 프로젝트의 모든 용어를 정의하고 일관성을 확보하기 위한 기준 문서입니다. 법적 리스크 회피를 위한 리브랜딩 용어, 도메인 용어, 기술 용어, 상태값을 포함합니다.

1.2 용어 변경 배경
법적 이슈 회피를 위해 금융 관련 용어를 커뮤니티 플랫폼 용어로 변경했습니다. "계모임"은 금융 관련 법규 저촉 가능성이 있고, "계주"는 금융 거래 주체로 오인될 수 있으며, "곗돈"은 상호금융법 저촉 가능성이 있습니다. 목표는 커뮤니티 챌린지 플랫폼으로 재포지셔닝하는 것입니다.

1.3 표기 원칙
팀 내부에서는 빠른 이해를 위해 "계모임"으로 통용하고, 외부 및 사용자에게는 법적 안전성을 위해 "챌린지"로 표기합니다. 문서 내에서는 혼용하되 괄호로 관계를 명시합니다(예: "챌린지(계모임)", "리더(계주)"). UI/API 응답에서는 "챌린지" 용어를 일관되게 사용합니다.

2.2 재화 및 거래 용어

| 우리두 어카운트   | Woorido        | 개인 가상 지갑 (가용 잔액 + 락 잔액)               | "총 잔액: 500,000원"                    |
|                  | Account        |                                                  |                                        |

7. 당도 시스템 (Brix System)
7.1 당도 개념
당도(Brix)는 사용자 신뢰도를 나타내는 지표입니다. 과일의 당도에 비유하여 "달콤한 관계"를 표현하며, 당도가 높을수록 신뢰할 수 있는 회원입니다.

7.2 당도 공식
최종 당도 = 기본 당도(12) + (납입 당도 × 0.7) + (활동 당도 × 0.15)

범위: 하한 없음 (마이너스 가능) ~ 상한 80
갱신 시점: 매월 1일 서포트 납입 시 (배치)
적용 범위: 유저 전체 통합 당도 (챌린지별 분리 X)
7.3 당도 항목
+----------------------+----------------+--------------------------------------------------+
| 항목                  | 당도            | 설명                                              |
+----------------------+----------------+--------------------------------------------------+
| [납입 당도]           |                |                                                  |
| 모임 참석             | +0.09/회       | ATTENDED 상태만 반영                              |
| 납입 개월             | +0.32/월       | 정상 납입 완료 기준                                |
| 연체                  | -1.5/회        | 납입 기한 초과 시                                  |
